# ni-investment-labour-market-2026
Northern Ireland Investment and Labour Markets Empirical Analysis
Master's Thesis, The Hertie School of Governance 2026
Author: Paul Schertzer
Advisor: Prof. Dr. Mark Hallerberg
What This Is
This repository contains the R script and data set for the empirical analysis in Chapter 5 of my Masters Thesis: Jobs for Peace? Inward Investment and Unemployment in Post-Agreement Northern Ireland
Evidence from 2002 - 2012
Research Question: Did inward investments in Northern Ireland contribute to reducing unemployment in the post-Agreement period relative to comparable UK regions?
Method
Cross-regional panel analysis, 2002-2012
